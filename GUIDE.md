## Часть 0: Что вообще происходит

**VPN** — это туннель между твоим устройством и сервером в другой стране. Весь интернет-трафик идёт через сервер, поэтому сайты видят IP сервера, а не твой.

**VPS** — это арендованный компьютер в дата-центре, на который мы поставим VPN-сервер. Стоит примерно $4-5/мес.

**AmneziaWG** — это технология VPN (протокол). Она маскирует трафик так, что блокировщики (DPI) не видят что это VPN, и пропускают. В РФ обычные VPN часто блокируют — AmneziaWG обходит.

**AmneziaVPN** — это приложение-клиент для подключения к AmneziaWG-серверу. Скачивается на твоё устройство (телефон / компьютер). Бесплатное.

⚠️ **Не путать с обычным WireGuard:** WireGuard — это другой протокол, его клиент к нашему серверу подключиться не сможет. Качай только **AmneziaVPN**.

Что мы будем делать:
1. Купим VPS (5 минут).
2. Поставим на него скрипт одной командой (1 минута).
3. Скачаем приложение **AmneziaVPN** на устройство.
4. Подключим телефон (1 минута через QR-код).
5. Подключим компьютер (3 минуты).

### 📥 Где скачать AmneziaVPN на устройство

Запомни эту таблицу — будем использовать в шагах 5 и 6:

| Устройство | Где скачать AmneziaVPN |
|------------|-----------------------|
| 📱 iPhone | [App Store](https://apps.apple.com/app/amneziavpn/id1600529900) |
| 📱 Android | [Google Play](https://play.google.com/store/apps/details?id=org.amnezia.vpn) |
| 💻 Windows / macOS / Linux | <https://amnezia.org/downloads> |

Иконка приложения — буква **«A»** на голубом/синем фоне.
⚠️ **НЕ скачивай:** WireGuard (зелёный дракон), OpenVPN, NordVPN, ExpressVPN — они не подключатся к нашему серверу.

## Часть 1: Купить VPS

### Где купить

Любой хостер подойдёт, главное: **локация НЕ в РФ** (иначе VPN бесполезен) и **виртуализация KVM** (не OpenVZ).

Проверенные варианты:
- **Hetzner** (Германия / Финляндия / США) : ~$4/мес, лучшее качество : <https://www.hetzner.com/cloud>
- **Aeza** (РФ-дружественный биллинг, серверы в ЕС) : <https://aeza.net>
- **FirstByte** (русский интерфейс) : <https://firstbyte.ru>

### Какой выбрать

Минимальный тариф:
- CPU: 1 ядро
- RAM: 1 GB
- Диск: 10 GB
- ОС: **Ubuntu 24.04** (или 22.04, тоже подойдёт)
- Локация: **Германия, Нидерланды, Финляндия, США**

### Что получишь после оплаты

В письме (или в личном кабинете хостера) будут 3 ключевых параметра:

```
IP-адрес сервера: 193.22.244.194    ← запиши, понадобится много раз
Логин:            root
Пароль:           AbCdEf123XyZ
```

## Часть 2: Подключиться к VPS по SSH

**SSH** — это инструмент для удалённого управления сервером через терминал. Ты вводишь команды у себя, они выполняются на сервере.

### На Windows

1. Открой меню Пуск → набери `PowerShell` → запусти.

> ![Скрин: запуск PowerShell в Windows](docs/screenshots/02-windows-powershell-start.png)

2. Введи команду (замени IP на свой):

```powershell
ssh root@193.22.244.194
```

3. В первый раз спросит:
   ```
   Are you sure you want to continue connecting (yes/no)?
   ```
   Введи **`yes`** (целиком слово, не `y`) и нажми Enter.

4. Спросит пароль:
   ```
   root@193.22.244.194's password:
   ```
   Введи пароль из письма хостера. **Буквы при вводе не отображаются — это нормально, не сломалось.** Просто вводи и жми Enter.

5. Если всё ОК, увидишь приветствие сервера:
   ```
   Welcome to Ubuntu 24.04.1 LTS ...
   root@your-server:~#
   ```

Поздравляю, ты на сервере!

### На macOS / Linux

То же самое, только PowerShell не нужен — используй встроенный **Terminal**:
- macOS: Cmd+Space → `Terminal`
- Linux: Ctrl+Alt+T

Команда та же:
```bash
ssh root@193.22.244.194
```

## Часть 3: Главное правило — где ты вводишь команду

**Это самая частая ошибка новичка.** Запомни сейчас, потом будет легче.

Терминал может быть в одном из двух режимов:

### 1. Локальный — ты на своём компе

Промпт выглядит так:
- Windows: `PS C:\Users\Сергей>`
- macOS/Linux: `сергей@MacBook ~ %` или `sergey@laptop:~$`

### 2. На сервере — ты внутри SSH-сессии VPS

Промпт выглядит так:
- `root@your-server:~#`
- `root@axxvaibssi:~#`
- В общем — **начинается с `root@что-то:~#`**

### Как переключаться

| Что хочу | Команда |
|----------|---------|
| Зайти на сервер | `ssh root@193.22.244.194` (на локальном компе) |
| Выйти с сервера | `exit` (находясь на сервере) |
| Прервать команду | `Ctrl+C` (не выходит из сессии, просто останавливает текущую команду) |

### Правило для команд в этой инструкции

Под каждой командой ниже я буду писать:
- 🖥️ **На сервере** : вводи когда промпт `root@что-то:~#` (после `ssh root@...`)
- 💻 **На своём компе** : вводи в PowerShell/Terminal **БЕЗ SSH**, промпт `PS C:\Users\...>` или `user@laptop`

Если ты внутри SSH и нужно «на своём компе» — сначала **`exit`**, потом команду.

## Часть 4: Установить VPN на VPS

### 4.1. Убедись что ты на сервере

Промпт должен быть `root@что-то:~#`. Если нет — подключись по SSH (Часть 2).

### 4.2. Выполни 3 команды по очереди

🖥️ **На сервере:**

```bash
apt update && apt install -y git
```

Подождёт ~30 секунд, выкачает обновления и установит `git`. В конце вернёт промпт.

```bash
git clone https://github.com/serenkiy-commits/amneziawg-quickstart.git
```

Скачает скрипты. ~5 секунд.

```bash
sudo bash amneziawg-quickstart/install.sh
```

Это главная команда. Она запустит установку.

### 4.3. Ответь на 3 вопроса

Скрипт спросит:

**Вопрос 1:**
```
Публичный IP сервера (например, 1.2.3.4) [193.22.244.194]:
```
→ **Нажми Enter.** Скрипт уже определил IP, в квадратных скобках.

**Вопрос 2:**
```
UDP-порт для VPN: число от 1 до 65535 [443]:
```
→ **Нажми Enter.** 443 — оптимальный порт (маскируется под HTTPS).

❌ **Не вводи путь к файлу.** Это вопрос про **число**, не про папку.

**Вопрос 3:**
```
Имя первого устройства (только буквы, цифры, дефис; например my-phone) [my-phone]:
```
→ Введи короткое имя для своего телефона: например **`my-phone`** или **`iphone`**. Или просто Enter (будет `my-phone`).

❌ Без пробелов, без точек, без `.conf`, без слешей. Только латиница, цифры, дефис.

### 4.4. Что увидишь дальше

Скрипт начнёт работу:
```
==> Устанавливаю AmneziaWG из PPA
✓ Пакеты установлены
✓ Модуль ядра amneziawg загружен
==> Включаю IP forwarding
✓ net.ipv4.ip_forward = 1
==> Генерирую ключи сервера и параметры обфускации
...
```

В конце нарисует **QR-код** в виде блоков и сводку:

```
═══════════════════════════════════════════════
  ✓ Готово
═══════════════════════════════════════════════

  Сервер:           193.22.244.194:443 (UDP)
  Интерфейс:        awg0
  ...
```

> ![Скрин: успешная установка с QR-кодом](docs/screenshots/03-install-success.png)

**Не закрывай это окно!** QR над сводкой — его сейчас будем сканировать.

## Часть 5: Подключить телефон

Самый быстрый способ. 1 минута.

### 5.1. Поставь приложение AmneziaVPN

⚠️ Именно **AmneziaVPN**, не «WireGuard» (он не подойдёт). Иконка — буква «A» на голубом фоне.

- 📱 iPhone: [App Store → AmneziaVPN](https://apps.apple.com/app/amneziavpn/id1600529900)
- 📱 Android: [Google Play → AmneziaVPN](https://play.google.com/store/apps/details?id=org.amnezia.vpn)

Если попадается в магазине несколько приложений с похожим названием — проверь разработчика, должен быть **AmneziaVPN** или **Privacy Accelerator**.

> ![Скрин: иконка AmneziaVPN в App Store / Google Play](docs/screenshots/04-amneziavpn-app-icon.png)

### 5.2. Открой приложение и добавь конфиг

1. В приложении нажми **«+»** (плюс) или **«Добавить сервер»**.
2. Выбери **«Сканировать QR-код»** (камера).

> ![Скрин: интерфейс AmneziaVPN, кнопка "Сканировать QR"](docs/screenshots/05-amneziavpn-scan-qr.png)

3. Наведи камеру телефона на QR-код в окне терминала на компьютере.
4. Приложение распознает и сохранит конфиг.

### 5.3. Подключись

Нажми **«Подключить»**. Через ~3 секунды появится индикатор «Подключено».

> ![Скрин: AmneziaVPN, состояние «Подключено»](docs/screenshots/06-amneziavpn-connected.png)

### 5.4. Если QR прокрутился вверх и не виден

🖥️ **На сервере** покажи QR заново:

```bash
qrencode -t ansiutf8 < /root/vpn-clients/my-phone.conf
```

Замени `my-phone` на имя, которое ты ввёл в шаге 4.3.

## Часть 6: Подключить компьютер

### 6.1. На сервере — создай отдельный конфиг для ноута

🖥️ **На сервере:**

```bash
sudo amneziawg-add-peer my-laptop
```

Скрипт создаст:
- `/root/vpn-clients/my-laptop.conf` — файл конфига
- `/root/vpn-clients/my-laptop.png` — QR-код картинкой

И покажет QR в терминале (для ноута он не нужен, можно проигнорировать).

### 6.2. Выйди из SSH-сессии

⚠️ **Это критически важно.** Файл нужно скачать **с компа на комп** через шифрованный канал. Внутри SSH этого делать нельзя.

🖥️ **На сервере** введи:

```bash
exit
```

Промпт сменится с `root@что-то:~#` на `PS C:\Users\Сергей>` (Windows) или `сергей@MacBook ~ %` (macOS). Теперь ты **на своём компе**.

### 6.3. Скачай файл на свой компьютер

💻 **На своём компе.** Команда зависит от ОС:

**Windows (PowerShell):**

```powershell
scp root@193.22.244.194:/root/vpn-clients/my-laptop.conf $env:USERPROFILE\Downloads\
```

**Замени `193.22.244.194` на свой IP сервера.** Угловые скобки `<>` НЕ нужны.

Спросит пароль root — введи. Файл появится в `C:\Users\Сергей\Downloads\my-laptop.conf`.

> ![Скрин: PowerShell, успешный scp](docs/screenshots/07-windows-scp-success.png)

**macOS / Linux:**

```bash
scp root@193.22.244.194:/root/vpn-clients/my-laptop.conf ~/Downloads/
```

Файл будет в `~/Downloads/my-laptop.conf`.

### 6.4. Альтернатива для Windows — без командной строки

Если scp не работает или непонятно:

1. Скачай **WinSCP**: <https://winscp.net/eng/download.php>
2. Установи, открой.
3. **Новое подключение** → введи:
   - Host name: `193.22.244.194` (твой IP)
   - Port: `22`
   - Username: `root`
   - Password: пароль root
4. Нажми **Login**, прими ключ.
5. Слева — твой компьютер, справа — сервер.
6. Справа зайди в папку `/root/vpn-clients/`.
7. Перетащи `my-laptop.conf` влево на свой Рабочий стол или Downloads.

> ![Скрин: WinSCP, две панели, перетаскивание .conf](docs/screenshots/08-winscp-drag.png)

### 6.5. Поставь AmneziaVPN на компьютер

Открой <https://amnezia.org/downloads> → выбери свою ОС → скачай **AmneziaVPN** (большая синяя кнопка). Установи как обычное приложение.

⚠️ Только **AmneziaVPN** с amnezia.org. Не качай WireGuard с wireguard.com — это другое приложение, оно не подключится к нашему серверу.

### 6.6. Импортируй конфиг

1. Открой AmneziaVPN.
2. **«Добавить сервер»** → **«Импорт конфигурации»** → **«Из файла .conf»**.
3. Выбери `my-laptop.conf` из Downloads.

> ![Скрин: AmneziaVPN на ПК, импорт .conf](docs/screenshots/09-amneziavpn-desktop-import.png)

4. Нажми **«Подключить»**.

## Часть 7: Проверка что VPN работает

### 7.1. Проверь IP

Открой в браузере: <https://2ip.ru>

Если показывает IP **твоего VPS** (193.22.244.194 или твой) — VPN работает.
Если показывает твой родной IP — VPN не активен или трафик идёт мимо туннеля.

> ![Скрин: 2ip.ru с IP сервера](docs/screenshots/10-2ip-success.png)

### 7.2. Проверь handshake (на сервере)

🖥️ **На сервере** (нужно снова зайти по SSH):

```bash
sudo awg show
```

Увидишь список пиров. У активного устройства должно быть свежее `latest handshake`:
```
peer: AbCdEf...
  endpoint: твой-внешний-IP:43210
  latest handshake: 15 seconds ago    ← свежий = работает
  transfer: 5.2 MiB received, 12.4 MiB sent
```

Если `(none)` или больше 3 минут — устройство не подключено или есть проблема.

## Часть 8: Если что-то пошло не так

### «Невалидный порт» или «Невалидное имя» во время установки

Ты ввёл не то значение в промпт скрипта (например, путь к файлу в поле порта).

**Решение:** запусти заново на сервере:
```bash
sudo bash amneziawg-quickstart/install.sh
```

На все вопросы можно жать Enter — этого достаточно.

### Подключился по SSH, но в ответ ничего, висит

Сервер недоступен или закрыт порт 22. Подожди 1 минуту (бывает на первой загрузке VPS). Если не помогло — проверь IP сервера в письме хостера.

### WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED

На этом IP раньше был другой сервер, Windows помнит его ключ. **Не атака.**

💻 **На своём компе:**
```powershell
ssh-keygen -R 193.22.244.194
```

Потом снова `ssh root@193.22.244.194`.

### scp пишет "Permission denied" или ничего не качает

Скорее всего ты ввёл команду **внутри SSH-сессии** (промпт `root@...:~#`). Сначала **`exit`**, потом scp.

Также проверь: в команде НЕ должно быть угловых скобок `<` и `>`. Должно быть `root@193.22.244.194:...`, не `root@<193.22.244.194>:...`.

### AmneziaVPN не импортит .conf

Убедись что ставишь именно **AmneziaVPN** (логотип Amnezia), а не **WireGuard** (логотип серого дракона). Они выглядят похоже, легко спутать.

Скачать только отсюда: <https://amnezia.org/downloads>

### Handshake идёт, но интернета нет

В панели хостера VPS открой исходящий UDP-трафик на порту 443 (или который выбрал). По умолчанию у некоторых хостеров firewall блокирует UDP.

### Подключено, но 2ip.ru показывает родной IP

Проверь конфиг на устройстве: `AllowedIPs = 0.0.0.0/0`. Если стоит `10.9.0.0/24` — туннель есть, но трафик в интернет идёт напрямую. Пересоздай конфиг через `add-peer` (свежие конфиги всегда с `0.0.0.0/0`).

### Сломал что-то совсем, хочу начать заново

🖥️ **На сервере:**
```bash
sudo bash amneziawg-quickstart/uninstall.sh
sudo bash amneziawg-quickstart/install.sh
```

Первая команда сносит VPN полностью, вторая ставит заново.
